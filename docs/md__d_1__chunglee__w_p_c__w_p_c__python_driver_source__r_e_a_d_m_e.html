pywpc is a Python-based driver for WPC devices.

This page is served as an application note for developers.

For a pure developer, prepare a Python environment with version at least above 3.8.

$ conda create --name cp310 python=3.10

For a release uploader, prepare 3 environments with Python 3.8, 3.9, & 3.10 each, named as:

  • cp38 for Python 3.8,
  • cp39 for Python 3.9, &
  • cp310 for Python 3.10.
    $ conda create --name cp310 python=3.10
    $ conda create --name cp39 python=3.9
    $ conda create --name cp38 python=3.8

In each of the environment(s), install required packages with:

$ pip install -r requirements.txt

For VS Code users, install C++ build tools in Visual Studio Installer.

  • Double-click upload_release.bat.
  • Press any button when it pauses.
  • Type PyPI username & password to authorize the upload.
  • This successively executes actions in the following section (Individual build).

Build example code by product

$ python setup.py example

Merge sources into a single file:

$ python setup.py merge

Clean documentation and regenerate .rst files:

$ python setup.py clean

Generate documentation:

$ python setup.py docs

Compile a pyd library:

$ python setup.py lib

Copying files to proper places:

$ python setup.py dist

Wrap up .pyd files:

$ python setup.py sdist build

Build a wheel:

$ python setup.py bdist_wheel

Upload pyd & wheels to PyPI platfrom:

$ twine upload dist/*
