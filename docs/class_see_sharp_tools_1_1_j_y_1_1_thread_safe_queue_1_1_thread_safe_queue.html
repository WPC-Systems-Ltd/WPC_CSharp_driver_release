Detailed Description

Thread Safe Queue is similar to Queue except Dequeue blocks when no elements are in queue until one is present or queue is destroyed..

Chinese Simplified: 安全线程队列是原始Queue的一种补充，当没有数据在队列中时，会在出队列中进行阻挡性操作，直到数据出现或者队列销毁。
