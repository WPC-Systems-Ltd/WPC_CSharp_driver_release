Open specific pin to digital output.

Parameters
portAvailable port number depends on WPC DAQ device.
pin_indexPin number you want to set. For example, you want to open pin 0, pin 1 and pin 5. pin_index = [0,1,5].
valuePin number you want to set high(1) or low(0). For example, set pin 0 and pin 1 to high, and pin 5 to low. value = [1,1,0].
timeoutSpecify an additional waiting time to set or get data.
Returns
0 if success, error otherwise.
